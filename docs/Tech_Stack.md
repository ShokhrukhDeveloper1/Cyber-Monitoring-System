# Technology Stack  

## Frontend  

- HTML  
- CSS  
- JavaScript  

Used to build the user interface and admin dashboard.  

## Backend  

- ASP.NET Core Web API (C#)  

Used to:  

- Handle user authentication  
- Process security data  
- Detect attacks  
- Manage blocked IPs  

## Database  

- SQL Server  

Used to store:  

- User information  
- Attack records  
- Security logs  
- Blocked IP addresses  

## Tools  

- Visual Studio  
- GitHub  
- Postman (for API testing)
